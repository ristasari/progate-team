Web ini dibuat untuk mengimplementasikan ilmu HTML dan CSS yang telah diberikan oleh Kominfo melalui kerja sama dengan mitra pelatihan Progate. Melalui program DTS ini diharapkan peserta mampu membuat halaman website secara mandiri.

Tentang DTS

Para peserta pelatihan dapat belajar dan latihan secara mandiri di Progate, dan dapat menanyakan materi pembelajaran kepada para Intruktur Profesional pada grup DTS Progam di Discord. Pada pelatihan ini juga terdapat project tim dengan tujuan agar para peserta pelatihan siap kerja dan menghadapi Revolusi Industri 4.0
